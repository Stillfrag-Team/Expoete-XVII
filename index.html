Sobre o Projeto

O projeto tabuleiro fantasma,consiste em uma modernização do antigo jogo da velha utilizando apenas de alguns recursos baratos.

O jogo da velha produzido para o projeto foi feito utilizando a ferramenta Unity em conjunto com o Visual Studio.O controle utilizado para jogar foi feito usando uma placa arduíno e um tabuleiro de xadrez.

Os Próprios alunos fizeram o jogo e seu controle,o jogo foi dividido entre as duas ferramentas para ser feito, o Visual Studio para programação e a Unity para o visual do jogo ser feito e para o jogo ser testado.O controle foi feito com nove botões, para selecionar os espaços e o jogo funcionar,sendo todos os botões e o tabuleiro personalizados, para ter uma temática de jogo da velha.

Inovação

A maior inovação do nosso projeto é a possibilidade de uma comunicação entre a engine de desenvolvimento de jogos Unity com Arduino uma coisa normalmente não muito simples de se fazer, que pode muito bem ser explorado em futuros projetos, caso você deseje saber como fizemos essa junção das duas ferramentes recomendamos esse site que tem um tutorial simplificado de facil entendimento e com codigos livres.

Projeto e Arduíno

Para criar esse projeto nós utilizamos do arduino da empresa Robocore que é a distribuidora oficial no Brasil.
