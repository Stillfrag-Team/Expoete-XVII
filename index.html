Sobre o Projeto

O projeto tabuleiro fantasma,consiste em uma modernização do jogo da velha com recuros baratos.

O jogo da velha produzido para o projeto foi feito utilizando a ferramenta Unity em conjunto com o Visual Studio.O controle utilizado para jogar foi feito usando uma placa arduíno e um tabuleiro de xadrez.

Os Próprios alunos fizeram o jogo e seu controle,o jogo foi dividido entre as duas ferramentas para ser feito, o Visual Studio para programação e a Unity para o visual do jogo ser feito e para o jogo ser testado.O controle foi feito com nove botões, para selecionar os espaços e o jogo funcionar,sendo todos os botões e o tabuleiro personalizados, para ter uma temática de jogo da velha.

Inovação

O controle feito é a maior inovação do projeto,por ele ser criado do zero é algo novo com potêncial para ser explorado em futuros projetos.
